Project DM-D.S.S

防災気象情報を提供するサービス

Project DM-D.S.S とは

Project DM-D.S.S は、個人によって運営されている、多様な防災気象情報を提供するサービスです。

公式サイト arrow_forward

特徴

  • 高速な情報配信

    WebSocket API により、リアルタイムで情報を取得できます。

  • 高い信頼性

    予報業務許可（地震動）を取得しており、気象庁のガイドラインに準拠して運営されているため、情報の信頼性が非常に高いです。

  • 多様な情報

    緊急地震速報、地震情報、津波情報、気象警報、天気予報など、多様な情報を提供しています。
