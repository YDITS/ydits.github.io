YDITS API

YDITS が提供する防災情報API

このAPIは現在開発段階です。いくつかのプロパティが不足しています。
現在利用できるエンドポイントは『緊急地震速報（警報）』のみです。

  • 緊急地震速報（警報） →
    Earthquake Early Warning (EEW) - Warning
  • 緊急地震速報（地震動予報） →
    Earthquake Early Warning (EEW) - Forecast
  • 地震情報 →
    Earhquake Information (eqinfo)
  • 津波情報 →
    Tsunami Information

サーバー１: 自宅サーバー (Node.js)
サーバー２: Cloudflare Workers
サーバー３: 自宅サーバー (ASP.NET Core)
