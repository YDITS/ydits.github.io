YDITS Project メンバー特約

この「YDITS Project メンバー特約」（以下「本規約」といいます。）は，よね/Yone（以下「当方」といいます。）が提供するサービス「YDITS」（以下「本サービス」といいます。）にて設置・提供している団体「YDITS Project」（以下「本団体」といいます。）の活動条件を、本団体に参加されたすべての方（以下「ユーザー」といいます。）との間で定めるものです。

第１条　本規約の適用範囲

  1. ユーザーは，本団体で活動するには本規約に同意しなければなりません。
  2. ユーザーは，本団体に参加した時点で本規約が適用されるものとします。
  3. ユーザーは，本規約の定めにしたがって本団体で活動するものとします。

第２条　禁止事項

本団体で活動するにあたって、以下の行為を禁止とします。

  1. YDITS利用規約 に反する行為

第３条　活動規則

本団体で活動するにあたって、以下の規則にしたがってください。

  • 技術課において

    1. ソフトウェア、ネットワーク、デバイス、その他技術的制作物の開発時にかかる費用等はご自身で負担いただきます。
    2. 団体として契約するサービス等は、必ず本部の許可を得てから行ってください。
    3. 法令等により制限されているコンテンツを制作しないでください。
    4. 他者になりすましてユーザーを誤解させるようなサービスは制作しないでください。
    5. 他者が作成したものを不正に使用、または誤った印象を与えるような形で使用しないでください。
    6. 虚偽のあるソフトウェア、悪意のあるソフトウェア、ネットワーク、デバイス、個人データを悪用または不正使用する意図のあるサービスは制作しないでください。
    7. マルウェアが含まれるソフトウェアを制作しないでください。
    8. デバイスにとって望ましくないソフトウェアを制作しないでください。
    9. 逆コンパイル、リバースエンジニアリング、もしくは逆アセンブル、またはソースコードの導出を行うようなソフトウェアを制作しないでください。
    10. ネットワーク、ソフトウェア、パスワード、暗号化コード、技術的保護手段のセキュリティを侵害、改ざん、もしくは回避するための手段に関与すること、またはこれらを他者が行うことを可能にすることはしないでください。

  • 配信課において

    1. 配信時にかかる電力費、デバイス費、ネットワーク費、および他にかかる費用等はご自身で負担いただきます。
    2. YouTubeにて配信を行う場合は、ライブ配信利用規約 および YouTube利用規約 にしたがって行ってください。
    3. 防災情報が一切含まれていない配信をしないでください。
    4. 他者が制作したものを不正に配信内で使用、または誤った印象を与えるような形で使用しないでください。
    5. 配信を通じて宣伝行為をしないでください。

  • 情報課において

    1. 他者が作成したものを不正に使用、または誤った印象を与えるような形で使用しないでください。
    2. 制作時にかかった費用等はご自身で負担いただきます。

その他、本部が不適切と判断した場合は措置を行います。

第４条　免責

  1. 当方は，本団体での活動によって生じた損害の責任は一切負わないものとします。
  2. 当方は，ユーザーに事前に通知することなく，本サービスの一部もしくは全部の内容を変更または停止できるものとし，それに伴うユーザーへのいかなる損害の責任は一切負わないものとします。

第５条　本規約の変更

  1. 当方は，ユーザーに個別の同意を得ることなく，本規約を変更することができるものとします。
  2. 当方は，本規約の変更時に，その内容についてユーザーに通知します。
