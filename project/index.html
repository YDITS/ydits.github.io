YDITS Project

YDITS Projectは当サービスが提供する防災活動団体です。

活動内容

サービス開発

利用者への防災サービス開発

ライブ配信

地震観測、地震監視など防災情報を動画サービスでライブ配信

その他

防災情報の提供

活動目標

社会に防災という形で貢献する

私たちは生活するなかで災害を経験します。 災害発生時に無くてはならない防災情報。 YDITSは防災という形で社会に少しでも貢献したい、そんな思いで生まれました。
その活動のなかで、多くの方の知識や視点、技術があることで、より質の高い防災情報を提供することが出来ると考え、このYDITS Project団体を設立しました。
本団体では、防災活動に少しでも関心のある方を募集しています。
