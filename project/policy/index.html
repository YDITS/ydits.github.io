YDITS Project プライバシーポリシー

YDITS Project（以下「当方」といいます。）は、当方の参加申請フォーム（以下、「本フォーム」といいます。）にて収集する、本フォームへ送信された方（以下、「ユーザー」といいます。）の個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます。）を定めます。

第１条　定義

「個人情報」とは、個人情報保護法における「個人情報」を指し、本フォームにて収集した、お名前、メールアドレス、その他の連絡先の情報を指します。

第２条　適用範囲

本ポリシーは、本フォームへ送信されたすべてのユーザーに適用されるものとします。

第３条　個人情報の利用目的及び用途

当方が個人情報を収集する目的は以下のとおりです。

  1. 当方からユーザーへ、申請いただいた内容についての返信を行うため。
  2. 年齢確認のため。
  3. 本人確認のため。

第４条　本ポリシーの更新

当方は、ユーザーに個別の同意を得ることなく、本ポリシーを更新することができるものとします。
